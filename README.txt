***This project contains code for solving the 8-puzzle problem.***

**Files Included:**
8_puzzle.py: The main script for solving the 8-puzzle.
testfile.txt: A test file containing example inputs to showcase how the code handles both valid and invalid puzzle configurations.

**How to Run:**
Make sure you are in the root directory of the project.
Run the following command in your terminal:

python3 8_puzzle.py testfile.txt

**Purpose of Files:**
8_puzzle.py: The main script that solves the 8-puzzle problem by reading puzzle configurations from testfile.txt and determining if each puzzle is solvable.
testfile.txt: A file containing multiple puzzle configurations for testing. It demonstrates how the code handles valid and invalid inputs, following the required input format.
